更新内容 #

1.本次只更新切图样式css名称的统一，关于javascript与各个模块的设计还在构思中，请先进行页面切图构造
2.每个分类都有起始ID,每个模块都有唯一的ID，ID是标记所有模块的唯一身份，每种业务场景都可以继续再往下进行分类，通用与产品、交互设计、前端和服务端开发

3.内置模块以"_"开始，写法上注意使用对象方法:
var _dropdown = {
uid:20300000,
version:"1.0.0",
init:function(){},
options:{},
events:{},
eventFn:{},
ajax:{}
}
其中每个对象都有init的方法，入参是一个对象，结构如下:
{
selector:_selector,//选择器模块
module:_module,//公共模块名称
options:_options,//所有该模块的公共附加对象
array:_retSelector//传入的新模块对象
}
具体DEMO请参照文件:"frontPage/js/alert.js"
开发模式下请将各自的js文件手动引入到页面头部第一个，可以参考"frontPage/alert.html"中的头部第一个js引入的例子

4.外置模块就采用Zr.add语法进行书写，路径是"./zr/*"
Zr.add("./zr/*",function(zr){
//todo something
},{requires:[]})

5.尽量不要更改公共模块：template.less、default.less、zr/index.js和其他公共属性，如果更改请在群内主动通知
6.CSS样式命名使用常见单词：show、hide、block、inner、outer。并且命名方式采用下划线命名方法：zr-dropdown-top-left,统一小写
7.浮层样式最高是9999,最小是100,其中警告提示信息类最高(9999)，下拉菜单等（2000）
~下拉菜单(zr-dropdown)：100~199
~导航菜单(zr-nav)：200~299
~下拉选择器(zr-select)：300~399
~模态框(zr-modal)：900~999
~通知提醒(zr-notification)：8100~8150
~全局提示(zr-message)：8050~8099
8.图片样式地址："//storage.jd.com/zrstatic/images/demo1.jpg"，"//storage.jd.com/zrstatic/images/demo2.jpg"，"//storage.jd.com/zrstatic/images/demo3.jpg"
